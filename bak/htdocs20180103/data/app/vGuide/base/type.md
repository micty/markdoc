##类目列表 GetProductTypeList
获取微商城类目列表。
两个特殊的一级类别：id=0表示上架未分类，id=-1表示未上架。
未分类为明细、无二级类别；如果商品类别（非微商城）有值，未上架为非明细；没有数据则为明细。


用途：商品列表中。

**参数：**`无`

**返回值：**
``` json
{
	"list":[
		{		
		"id":"商品类别id",
		"code":"代码",
		"name":"名称",
		"detail":"是否明细。1：明细，0：不是明细 (int)",
		"parentid":"父类别id：0_一级类别",
		"level":"类别级别：只存在两级1和2"
		}
	]
}
```
没有数据：
``` json
{
	"list":[]
}
```
