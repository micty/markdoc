##购物车商品信息 GetCartInfo
获取购物车商品信息。


用途：加载购物车。

**参数：**

名称 |  描述
------| ------
`productIds` | 商品id数组(商品列表返回的id)

**返回值：**


``` json
{	
 "list":[
		{		
		"id":"商品id",
		"name":"商品名称",
		"type":"商品类型：0：普通商品，1：合并商品，2：有辅助属性商品",
		"unit":"单位",
		"unitId":"单位id",
		"image":"图片url",
		}
	]
}
```
